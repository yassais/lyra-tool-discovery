# Changelog

All notable changes to LLMs Forge will be documented in this file.

## [1.0.0] - 2026-01-15

### Added

- Initial release of LLMs Forge
- Web interface for extracting llms.txt documentation
- Automatic discovery of llms.txt and llms-full.txt files
- Document parsing by markdown headers
- Generation of consolidated llms-full.md
- Generation of AGENT-GUIDE.md for AI assistants
- Individual file downloads
- ZIP bundle download
- MCP (Model Context Protocol) server
- REST API for programmatic access
- Material for MkDocs documentation site

### Features

- Smart URL handling (auto-adds https://)
- Token estimation for all documents
- Processing time tracking
- Mobile-responsive UI
- Dark mode support
- Copy to clipboard functionality

## [Unreleased]
